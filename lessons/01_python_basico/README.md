# Lección 01: Python básico

Objetivo: variables, listas, diccionarios y funciones aplicados a “precios”.

## Cómo ejecutar

Desde la raíz del proyecto (con el entorno virtual activado):

```bash
python lessons/01_python_basico/ejercicio.py
```

O desde esta carpeta:

```bash
python ejercicio.py
```

## Contenido

- `ejercicio.py`: lista de precios, funciones para media y máximo, diccionario de resumen.
- Después de ejecutar, modifica los precios o añade más y vuelve a ejecutar para practicar.
